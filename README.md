Project Description
-------------------
Duka Bora is a PHP and MySQL based Inventory Management System designed to help small businesses manage products, suppliers, sales and inventory.

Technologies Used
-----------------
- PHP
- MySQL
- HTML5
- CSS3
- Laragon
- phpMyAdmin
- Visual Studio Code

Features
--------
- Add Product
- Edit Product
- Delete Product
- View Products
- Record Sales
- Sales History
- Inventory Report
- Low Stock Alert
- Best Selling Products Report
- Last Viewed Product Cookie

Database
--------
Database Name:
duka_bora

Import:
database/duka_bora.sql

How to Run
----------
1. Install Laragon.
2. Copy the project into Laragon/www/.
3. Start Apache and MySQL.
4. Import database/duka_bora.sql into phpMyAdmin.
5. Open:
http://localhost/duka_bora
